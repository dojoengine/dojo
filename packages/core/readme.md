# Dojo Core SDK

Dojo core package to interact with Dojo worlds. It exposes World interface commands to allow the creation of:

- World explorers
- World deployers
- Dojo games
- Anaylitics 

### Get started

```
yarn add @dojoengine/core
```

### Account

Starknet.js Account wrapper for quick testing
